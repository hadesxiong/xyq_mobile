      泗塘污水处理厂初雨调蓄工程主要建设内容包括：一是拆除1座二沉池配水井和4座二沉池，新建一座全地下式初雨调蓄池规模11000立方米；二是实施建筑、电气、自控和除臭等工程及道路、绿化、围墙等附属设施；三是新建雨水截流管管径为DN2000。新建地下调蓄池总建筑面积1210平方米，其中地上建筑面积385平方米、地下建筑面积为825平方米。


      泗塘污水处理厂初雨调蓄工程是继长桥污水处理厂后，第二个具备通水条件完成环保督察考核节点的“六厂”项目。项目建成后，泗塘区域初期雨水可暂留调蓄池内，待旱季后再从调蓄池中将雨水慢慢地排出，既能提高雨水利用率，又能控制初期雨水对受纳水体的污染，项目有利于水域的整体水体环境质量提升，改善城乡环境，保障区域河道水质，保证社会和经济可持续发展及人民正常生活，提升上海城市的整体形象。
